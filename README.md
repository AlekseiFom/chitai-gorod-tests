# Автотесты для сайта «Читай-город»

## 📌 Описание проекта

Данный проект представляет собой фреймворк для автоматизированного тестирования интернет-магазина **«Читай-город»** ([chitai-gorod.ru](https://www.chitai-gorod.ru)).

Проект разработан в рамках дипломной работы по тестированию программного обеспечения и включает:
- **API-тесты** — проверка бэкенда через HTTP-запросы.
- **UI-тесты** — проверка пользовательского интерфейса через браузер (Selenium WebDriver).

---

## 🛠️ Технологический стек

| Компонент | Технология |
|-----------|------------|
| Язык программирования | Python 3.10+ |
| Фреймворк тестирования | pytest |
| UI-тесты | Selenium WebDriver |
| API-тесты | Requests |
| Отчёты | Allure |
| Управление зависимостями | pip + requirements.txt |
| Хранение секретов | python-dotenv (.env файл) |
| Линтер | flake8 (PEP8) |
